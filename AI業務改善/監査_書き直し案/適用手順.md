# 書き直し案の適用手順（承認後にやること）

このフォルダの中身は**案**です。あなたが承認するまで、元のファイルは一切変更していません。

## 承認したら（3分で終わります）

AIセッションに次のように指示すれば適用されます：

```
AI業務改善/監査_書き直し案/ の内容を承認します。
1. CLAUDE.md をリポジトリ直下へ配置
2. SKILL.md を .claude/skills/mitsumori-design/SKILL.md へ配置し、元の DESIGN.md を削除
3. 見積書作成.txt は【削除する／「旧版」と明記して残す】← どちらか選んで書く
を実行してコミット・プッシュしてください。
```

手動でやる場合：

```bash
cp AI業務改善/監査_書き直し案/CLAUDE.md ./CLAUDE.md
mkdir -p .claude/skills/mitsumori-design
git mv DESIGN.md .claude/skills/mitsumori-design/SKILL.md   # その後、案の+1行を反映
```

## 元に戻したくなったら

すべてgitに残っています： `git show d68792f:DESIGN.md > DESIGN.md`
